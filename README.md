# 🧠 Fake News Detection System

A machine learning-based web application that detects whether a news article is **REAL or FAKE** using Natural Language Processing (NLP).

---

## 🚀 Live Demo

👉 https://srkdev022.streamlit.app/

Scan QR or open the link to try the app in real-time.

---

## ✨ Features

* 🔍 Real-time fake news prediction
* 📊 Confidence score & risk analysis
* 🧠 NLP-based text processing
* 🎯 High accuracy (~99.4%)
* 🌐 Deployed using Streamlit

---

## 🧠 Tech Stack

* Python
* Scikit-learn
* TF-IDF Vectorization
* Logistic Regression
* Streamlit

---

## 📊 Model Performance

* ✅ Accuracy: **99.4%**
* 🔧 Hyperparameter tuning using GridSearchCV
* 📈 Model comparison:

  * Logistic Regression
  * Naive Bayes
  * SVM

---

## ⚙️ How It Works

1. Input news text
2. Text cleaning & preprocessing
3. Convert text → numerical features (TF-IDF)
4. Model predicts REAL or FAKE
5. Displays confidence & risk score
